## Instructions

Scripts are provided to build and run the project for Windows, macOS, and linux.

Prerequisites:
Maven and Java must be installed

=======
To run:

macOS and linux:  
`cd QuizCreator`  
`./build.sh`  
`./run.sh`  

windows
`cd QuizCreator`  
`./build.bat`
`./run.bat`

After successfully building and running, you can view the project by going to:
http://localhost:8080
